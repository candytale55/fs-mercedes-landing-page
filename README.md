# Mercedes-Benz Landing Page

A responsive landing page for Mercedes-Benz.

## Structure

- `index.html` - Main HTML file
- `src/` - Source files
  - `css/` - Stylesheets
  - `js/` - JavaScript files
  - `images/` - Image assets
- `docs/` - Documentation

## Getting Started

Open `index.html` in your browser to view the landing page.
